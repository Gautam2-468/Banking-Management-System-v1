# Banking-Management-System-v1
A simple Banking Management System built using Python, OOP, and File Handling.
Features
Deposit Money
Withdraw Money
Check Account Balance
Change Password
Transaction History
Password Verification
Data Storage using Text Files
Technologies Used
Python
OOP Concepts
File Handling
Project Structure
banking_management.py - Main source code
balance.txt - Stores account balance
password.txt - Stores account password
History.txt - Stores transaction history
Default Account Details
Account Number: 12345
Password: 123
